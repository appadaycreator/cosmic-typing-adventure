プライバシーポリシー

お客様の個人情報保護とプライバシーの安全性について

最終更新日：2025年7月12日

基本方針

Cosmic Typing Adventure（以下「当サービス」）では、ユーザーの皆様に安心してサービスをご利用いただくため、 個人情報の保護を最重要課題として取り組んでおります。

当サービスは、個人情報保護法及び関連法令を遵守し、適切な個人情報の取り扱いを行います。

収集する情報

個人情報

  • • メールアドレス（アカウント作成・認証用）
  • • ユーザー名（表示名）
  • • プロフィール画像（任意）

利用情報

  • • タイピング練習記録（WPM、正確率、練習時間など）
  • • ゲーム進行状況（レベル、実績、探索記録など）
  • • 設定情報（言語、フォントサイズ、通知設定など）
  • • アクセスログ（IPアドレス、ブラウザ情報、アクセス時刻など）

情報の利用目的

サービス提供

  • • アカウント管理
  • • ゲーム進行の保存
  • • 統計データの表示
  • • カスタマイズ機能

サービス改善

  • • 機能の最適化
  • • バグの修正
  • • 新機能の開発
  • • ユーザー体験の向上

コミュニケーション

  • • 重要なお知らせ
  • • サポート対応
  • • アップデート通知
  • • セキュリティ通知

セキュリティ

  • • 不正利用の防止
  • • セキュリティ監視
  • • 法的要請への対応
  • • システム保護

Cookie・ローカルストレージの使用

Cookieについて

当サービスでは、より良いユーザー体験を提供するためにCookieを使用しています。 Cookieは、ウェブサイトがお客様のコンピュータに保存する小さなテキストファイルです。

使用目的

必須Cookie

ログイン状態の維持、セッション管理

機能Cookie

設定の保存、言語・テーマ設定

解析Cookie

利用状況分析、サービス改善

Cookie設定

ブラウザの設定によりCookieを無効にすることができますが、 一部機能が制限される場合があります。

データの管理・保護

データ保存

当サービスは、Supabase（米国PostgreSQL基盤のBaaS）を使用してデータを安全に管理しています。

  • • データはSSL/TLS暗号化により保護
  • • 定期的なバックアップを実施
  • • アクセス制御により不正アクセスを防止
  • • データの完全性を保証

セキュリティ対策

パスワードの暗号化保存
二要素認証対応
定期的なセキュリティ監査
アクセスログの監視
不正アクセス検知システム
データの最小権限アクセス

第三者への情報提供

基本原則

当サービスは、法令に基づく場合や、お客様の同意がある場合を除き、 個人情報を第三者に提供することはありません。

提供する場合

法的要請

裁判所命令、法執行機関からの正式な要請

緊急時

生命、身体、財産の保護に必要な場合

同意取得

事前にお客様の明示的な同意を得た場合

お客様の権利

お客様の個人情報に関して、以下の権利を保障いたします：

開示請求権

保存されている個人情報の確認

訂正・削除権

間違った情報の修正・削除

利用停止権

個人情報の利用・処理の停止

データポータビリティ

データの移行・ダウンロード

アカウント削除権

すべての個人情報の完全削除

異議申立権

データ処理に対する異議申立

権利行使について

これらの権利を行使される場合は、お問い合わせフォームまたは こちら からご連絡ください。本人確認後、適切に対応いたします。

プライバシーポリシーの変更

当サービスは、法令の変更やサービスの改善に伴い、 このプライバシーポリシーを変更する場合があります。

軽微な変更

  • • ウェブサイトでの掲載により通知
  • • 変更日から効力発生
  • • 継続利用により同意とみなす

重要な変更

  • • 事前のメール通知
  • • 十分な検討期間を提供
  • • 明示的な同意を取得

変更内容は当ページで確認できます。定期的にご確認いただくことをお勧めします。
