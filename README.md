# Cosmic Typing Adventure 🚀

宇宙をテーマにしたタイピング練習アプリケーション。美しい惑星の背景で楽しくタイピングスキルを向上させましょう！

## 🌟 特徴

- **完全な日本語タイピングエンジン**: 全ひらがな・カタカナ・濁音・半濁音・拗音・促音対応
- **柔軟なローマ字入力**: 複数の入力パターン対応（例: "し" → "shi" or "si" or "ci"）
- **リアルタイム入力候補表示**: 現在の入力と可能なパターンを画面下部に表示
- **高精度なミス判定**: 誤入力を即座に検出して赤色でハイライト
- **多彩なゲームモード**: 通常練習、サバイバル、タイムアタック（30秒/1分/3分/5分）
- **ランク評価システム**: S/A/B/C/Dランクで実力を可視化
- **リーダーボード**: モード別のハイスコアランキング
- **宇宙テーマ**: 美しい惑星背景とアニメーション
- **リアルタイム統計**: タイピング速度、精度、進捗をリアルタイムで表示
- **パフォーマンスグラフ**: Chart.jsを使用した詳細な統計分析
  - WPM推移グラフ（折れ線グラフ）
  - 正確率推移グラフ（折れ線グラフ）
  - 苦手キーの棒グラフ（トップ15表示）
  - 練習時間の円グラフ（日別/週別/月別）
  - データエクスポート機能（CSV/JSON）
- **Supabase統合**: クラウドベースのデータ保存と同期
- **レスポンシブデザイン**: モバイル、タブレット、デスクトップ対応
- **アクセシビリティ**: スクリーンリーダー対応とキーボードナビゲーション
- **多言語対応**: 日本語と英語のインターフェース
- **オフライン対応**: ローカルストレージによる完全オフライン機能

### 🎮 ゲームモード (v3.1.0)

#### サバイバルモード
- **ライフシステム**: ミス3回で終了
- **ライフ表示**: ❤️アイコンでリアルタイム表示
- **シェイクアニメーション**: ミス時の視覚的フィードバック
- **ランク評価**: 結果に応じてS/A/B/C/Dランクを付与
- **リーダーボード**: 上位10位のハイスコア表示

#### タイムアタックモード
- **4つの制限時間**: 30秒スプリント / 1分クイック / 3分マラソン / 5分エンドレス
- **カウントダウンタイマー**: 残り時間をリアルタイム表示
- **リアルタイムWPM更新**: 入力速度を秒単位で更新
- **ランク評価システム**:
  - **Sランク**: スコア80以上（WPM × 正確率/100）
  - **Aランク**: スコア60-79
  - **Bランク**: スコア40-59
  - **Cランク**: スコア20-39
  - **Dランク**: スコア20未満
- **詳細な結果表示**: WPM、正確率、入力文字数、使用時間、CPM
- **リーダーボード**: 制限時間別のハイスコアランキング

#### リーダーボード機能
- **モード別ランキング**: サバイバル、タイムアタック（時間別）
- **上位10位表示**: トッププレイヤーのスコア一覧
- **自動保存**: Supabaseとローカルストレージの二重保存
- **オフライン対応**: ネットワーク切断時もローカルデータで動作
- **プレイヤー名**: カスタマイズ可能（デフォルト: Anonymous）

### 🆕 練習テキスト管理システム (v2.0.0)

- **200+練習テキスト**: 4カテゴリ（日常会話・ビジネス・プログラミング・文学）× 50文章以上
- **難易度別フィルタリング**: 初級・中級・上級の3段階で適切なテキストを選択
- **カテゴリ別分類**: 目的に応じたテキスト選択が可能
- **お気に入り機能**: よく使うテキストをブックマーク
- **カスタムテキスト**: ユーザー独自のテキストを追加可能
  - 手動入力
  - ファイルアップロード（.txt, .md）
  - URL読み込み（Webページから自動抽出）
- **習熟度に応じた自動出題**: ユーザーの平均WPMと正確率から最適な難易度を推薦
- **テキスト統計**: 文字数、単語数、使用頻度を自動計算
- **Supabase連携**: クラウドでのテキスト管理と同期

## 🎯 使用方法

### 基本的なタイピング練習
1. 惑星を選択
2. 表示されるテキストを入力
3. リアルタイムで統計を確認
4. 結果を保存（オプション）

### 統計の確認
- **WPM**: 1分間の単語数
- **CPM**: 1分間の文字数
- **精度**: 正確性の割合
- **進捗**: 時間経過でのパフォーマンス

### 統計グラフ機能
1. **パフォーマンス推移グラフ**
   - WPMと正確率の折れ線グラフ
   - 直近30セッションのデータを表示
   - 2軸グラフでWPMと正確率を同時表示

2. **苦手キー分析グラフ**
   - エラー回数の多いキーを棒グラフで表示
   - トップ15の苦手キーを可視化
   - 平均エラー数もツールチップで確認可能

3. **詳細分析タブ**
   - 日別統計: 最新30日間のデータ
   - 週別統計: 最新12週間のデータ
   - 月別統計: 最新12ヶ月のデータ
   - キー分析: エラー頻度とセッション数
   - 進歩トレンド: 最新30セッションの推移

4. **データエクスポート**
   - CSV形式: 日別データをスプレッドシートで分析
   - JSON形式: すべての分析データをバックアップ

### 設定オプション
- テーマ切り替え
- 音效設定
- 統計表示オプション
- キーボードショートカット
